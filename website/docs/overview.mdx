Spotifly facilitates some common interactions with Spotify's API and services.

### [@spotifly/core](/docs/packages/core)

![npm (scoped)](https://img.shields.io/npm/v/@spotifly/core) ![Codecov](https://codecov.io/gh/eegli/spotifly/branch/main/graph/badge.svg?flag=core)

A lightweight, strongly typed wrapper for the Spotify Web API.

- Command Line Interface ❌
- Programmatic use ✅

### [@spotifly/auth-token](/docs/packages/auth-token)

![npm](https://img.shields.io/npm/v/@spotifly/auth-token) ![Codecov](https://codecov.io/gh/eegli/spotifly/branch/main/graph/badge.svg?flag=auth-token)

Easily retrieve Spotify authentication/refresh tokens that are suitable to run in independent apps.

- Command Line Interface ✅
- Programmatic use ✅

### [@spotifly/library](/docs/packages/library)

![npm](https://img.shields.io/npm/v/@spotifly/library) ![Codecov](https://codecov.io/gh/eegli/spotifly/branch/main/graph/badge.svg?flag=library)

Export your enriched Spotify user library to JSON.

- Command Line Interface ✅
- Programmatic use ❌
