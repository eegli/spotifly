Spotifly facilitates some common interactions with Spotify's API and services.

### [@spotifly/core](/docs/packages/core) ![npm](https://img.shields.io/npm/v/@spotifly/core)

A lightweight, strongly typed wrapper for the Spotify Web API.

- Command Line Interface ❌
- Programmatic use ✅

### [@spotifly/auth-token](/docs/packages/auth-token) ![npm](https://img.shields.io/npm/v/@spotifly/auth-token)

Easily retrieve Spotify authentication/refresh tokens that are suitable to run in independent apps.

- Command Line Interface ✅
- Programmatic use ✅

### [@spotifly/library](/docs/packages/library) ![npm](https://img.shields.io/npm/v/@spotifly/library)

Export your Spotify user library to JSON.

- Command Line Interface ✅
- Programmatic use ❌
