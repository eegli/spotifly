Library

Export your Spotify library to JSON.

You will need an access token from Spotify. You can get one from right here!

CLI Usage‚Äã

Create your custom command using the query builder below and paste it into your terminal.

token¬†*
light
type
outDir

With npx

npx @spotifly/library@latest --token token --type light

With cli

spotifly library --token token --type light

Options‚Äã

Flag(required?) Description
--token‚úÖ A Spotify user access token. Requires at least the scope "user-library-read"
--type‚ùå Output type per track. Either full or light. Default: light
--genres‚ùå Include artist genres for each track. Default: false
--features‚ùå Include audio features for each track. Default: false
--compact‚ùå Output more compact/minified JSON and save disk space. Default: false
--outDir‚ùå Custom relative output directory. Default: Current directory

Output‚Äã

A light track has a reduced set of properties:

type TrackLight = {
  id: string;
  name: string;
  album: {
    name: string;
    id: string;
  };
  artists: {
    id: string;
    name: string;
  }[];
};

The full has all the track properties from the Spotify API. The next page features an example of both outputs.
