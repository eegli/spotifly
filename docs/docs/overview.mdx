Spotifly facilitates some common interactions with Spotify's API and services.

## Packages

The following packages are available:

### [@spotifly/packages/auth-token](/docs/packages/auth-token) ![npm](https://img.shields.io/npm/v/@spotifly/auth-token)

Easily retrieve Spotify authentication/refresh tokens that are suitable to run in independent apps.

- Command Line Interface ✅
- Programmatic use ✅

### [@spotifly/library](/docs/packages/library) ![npm](https://img.shields.io/npm/v/@spotifly/library)

Export your Spotify user library to JSON.

- Command Line Interface ✅
- Programmatic use ❌

---

All packages that support a CLI can either be used with Spotifly's wrapper CLI or as standalone executable.
