Overview

Spotifly facilitates some common interactions with Spotify's API and services.

@spotifly/core​

A lightweight, strongly typed wrapper for the Spotify Web API.

  • Command Line Interface ❌
  • Programmatic use ✅

@spotifly/auth-token​

Easily retrieve Spotify authentication/refresh tokens that are suitable to run in independent apps.

  • Command Line Interface ✅
  • Programmatic use ✅

@spotifly/library​

Export your enriched Spotify user library to JSON.

  • Command Line Interface ✅
  • Programmatic use ❌

@spotifly/cli​

The Spotifly CLI.

All packages are provided as-is. It is in your responsibility to make sure your applications is compliant with the Spotify Developer Terms.
